# Working with Amazon OpenSearch Service - AWS OpenSearch

### Overview

In this workshop, you will learn how to integrate Amazon OpenSearch with a serverless backend using AWS Lambda and deploy a simple static website on AWS using S3 and CloudFront. The workshop also covers how to log request data for each search query to analyze search keywords and request status using CloudWatch.
